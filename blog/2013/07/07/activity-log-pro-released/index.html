Activity Log Pro Released

Jul 7th, 2013

The first version of Activity Log Pro Android app is now available! Here’s a quick overview of the new features:

  • No Ads!
  • Calendar view of entries
  • Export data as an Excel spreadsheet (CSV file)
  • Automatic backup of data to the cloud
  • Options for the order in which activities are displayed
  • Star/favorite activities for easy filtering
  • Updated UI with Android 4.x look and feel and support for high resolution displays

And there are plans to add more in the future! So, do give it a spin!

Usability improvements

The Pro version while retaining the flexibility of the free version significantly improves usability in multiple ways. Most signicantly, entries are now also displayed on a monthly calendar to easily visualize the regularity with which an activity is performed. Activities can now be “starred” with an option to show only Starred activities. This replaces the Archive function found in the free version. There are minor UI improvements all over and the look-and-feel has been updated to be consistent with Android 4.x. The app looks great on the HD displays on our Galaxy Nexus phone and Nexus 7 tablet test devices!

Behind the scenes

Android doesn’t have a native calendar widget to display data, so we had to roll our own. While it was an interesting exercise, this should be something that the framework should include. On the other hand, implementing automatic backup of the data to the cloud was comparatively straightforward thanks to the almost magical Android Backup Service. The Backup Service is only available on Android 2.2 and later which means the Pro version isn’t available on pre-2.2 devices. Which is not a big deal given that pre-2.2 devices account for less than 1% of the installation base of the free version (which, by the way, works for Android 1.6 and later!). The excellent ActionBarSherlock library made it super-easy to have an action bar that will work for pre-Honeycomb devices. In our opinion, it should be bundled with the Android support libary!

Future plans

While we will continue to support the free version with bugfixes and the ocassional feature addition, the Pro version is almost a new (and better!) app in its own right and we plan to focus on it to make it even better.
